# Udacity-Capstone_Project_Travel_app

### The goal of this project

- Setting up Webpack
- babel
-- OOP for Trip
- Interact with Restful APIs
    - weather, location, picuters and countries 
- Create full informative trip for UI
- TO DO LIST APP 


### Technologies 
1. JS 
2. HTML
3. Scss
4. express 
5. Webpack
6. worker services



### TO start 
```
npm i
npm run prod/dev
npm run start 
```
